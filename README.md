# Nova-Calculator
A premium desktop calculator designed with a strong emphasis on user experience, visual polish, performance, and maintainable architecture.
Unlike traditional browser calculators, Nova Calculator combines a refined interface with advanced mathematical functionality, responsive layouts, smooth animations, and accessibility-first design principles.
## Features

### Standard Calculator

- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Parentheses
- Decimal calculations
- Continuous expressions

### Scientific Functions

- Square
- Cube
- Powers
- Square root
- nth root
- Logarithm
- Natural logarithm
- Exponential
- Factorial
- Trigonometric functions
- Hyperbolic functions
- Absolute value
- ПЂ
- Euler's number
- Random numbers

### Memory Operations

- MC
- MR
- MS
- M+
- M-

### Productivity

- Calculation history
- Persistent Local Storage
- Copy results
- Search history
- Favorite calculations
- Undo / Redo

### User Experience

- Responsive design
- Dark and Light themes
- Keyboard shortcuts
- Smooth animations
- Beautiful transitions
- Glassmorphism interface
- Accessible controls
- Error handling
- Expression validation

## Installation

Clone the repository

```bash
git clone https://github.com/username/nova-calculator.git
```

Navigate into the project

```bash
cd nova-calculator
```

Open

```
index.html
```

or

```bash
npm install

npm run dev
```
